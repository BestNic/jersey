# BrainShop - Static Frontend

Static mini e-commerce showcase for football jerseys with WhatsApp ordering.

## Local preview

Open `index.html` directly in your browser, or run a simple static server:

```bash
python -m http.server 5500
```

Then visit `http://localhost:5500`.

## Deploy

- **GitHub Pages:** upload this folder as a repository and enable Pages from branch root.
- **Vercel:** import the folder/repository and deploy as a static site (no build command required).
